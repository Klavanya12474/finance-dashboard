# 💰 Finance Dashboard

A simple and interactive Finance Dashboard built using React. This app helps users track their balance, add transactions, and manage expenses easily.

---

## 🚀 Features

- 📊 Dashboard overview
- 💳 Add transactions (income/expense)
- 💰 Real-time balance update
- 🧾 Transaction history
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (ES6)
- CSS3

```bash

cd finance-dashboard
Install dependencies
npm install
Run the development server
npm run dev
Open in browser
